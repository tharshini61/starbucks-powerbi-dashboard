# Starbucks Customer Behavior Analysis

## Project Overview
This project analyzes customer behavior using the Starbucks dataset and visualizes insights in Power BI.

## Dataset
- Profile Dataset
- Portfolio Dataset
- Transcript Dataset

## Tools Used
- Python
- Google Colab
- Power BI
- GitHub

## Data Cleaning
- Checked missing values
- Checked duplicate records
- Removed unnecessary columns
- Analyzed age, income, gender, offer type, and event data

## Dashboard Insights
- Customer demographics
- Gender distribution
- Income analysis
- Offer type analysis
- Event analysis
- Membership trends
